# Portfolio Website

A modern and responsive portfolio website for a Full Stack Software Engineer built with React and CSS.

## Features

- Clean hero section with name, title, and introduction
- About Me section
- Skills section showcasing technical expertise
- Projects section with 3 sample projects
- Contact section with email and LinkedIn links
- Fully responsive design for mobile and desktop
- Modern, clean, and professional styling

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Customization

- Update personal information in the component files:
  - `src/components/Hero.jsx` - Name and introduction
  - `src/components/About.jsx` - About me content
  - `src/components/Skills.jsx` - Skills list
  - `src/components/Projects.jsx` - Projects information
  - `src/components/Contact.jsx` - Contact email and LinkedIn

- Modify colors and styling in the CSS files located in each component's directory.
